# 🏖️ Leisure Travel Planner Skill

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.0.0-blue.svg)]()

> 面向旅游爱好者的AI休闲出行规划 - 景点打卡路线 + 美食推荐 + 拍照点 + 攻略生成

## ✨ Features

- 🏞️ 景点推荐与评分（高德POI + 小红书攻略）
- 📍 打卡路线优化（地理位置就近原则）
- 📸 拍照打卡点推荐（网红地 + 最佳角度）
- 🍜 美食深度推荐（当地特色 + 必吃榜）
- 🏨 全平台住宿比价（酒店/民宿/青旅）
- 🚄 实时交通（12306 + 高德 + 航班）
- 💰 三档预算（穷游/舒适/奢华）
- 📱 HTML报告生成
- 🌤️ 天气穿搭建议
- 🧳 旅行清单自动生成

## 🆚 vs 商旅出行

| | 商旅出行 | 休闲出行（本Skill） |
|--|---------|-------------------|
| 目标 | 高效到达 | 体验最大化 |
| 特色 | 差旅合规 | 打卡路线优化 |
| 美食 | 商务宴请 | 深度当地特色 |
| 独有 | 报销凭证 | 拍照点/Vlog攻略 |
