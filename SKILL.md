# 休闲出行规划技能 (Leisure Travel Planner)

> **面向旅游爱好者的AI休闲出行规划能力**
> 开源版本 v1.0 · 2026-04-09 · MIT License

---

## 一、技能概述

本技能为旅游爱好者提供**一站式智能休闲出行规划 + 打卡路线优化 + 美食推荐**服务。

**v1.0 核心特性**：
- 🏞️ **景点推荐与评分**：高德POI + 小红书攻略 + 评分排序
- 📍 **打卡路线优化**：基于地理位置的最优游览顺序规划
- 📸 **拍照点推荐**：网红打卡地 + 最佳拍照时间/角度建议
- 🍜 **美食深度推荐**：当地特色美食 + 餐厅推荐 + 菜品必点
- 🚄 **实时交通**：12306 MCP + 高德路径规划 + 航班搜索
- 🏨 **住宿推荐**：全平台比价（携程/飞猪/Booking/Agoda/民宿）
- 💰 **旅行预算**：穷游/舒适/奢华三档标准
- 📱 **HTML报告**：精美HTML页面 + 真实预订链接
- 📑 **PDF导出**：支持PDF格式输出
- 📝 **飞书云文档**：检测到飞书MCP时自动生成
- 🔄 **跨平台兼容**：自动降级输出（全格式→HTML→Markdown）
- 🌤️ **天气穿搭**：目的地天气 + 穿衣建议
- 🧳 **旅行清单**：行李/证件/药品/电子设备自动生成

**触发条件**：
- "帮我规划旅游" / "出去玩" / "度假攻略"
- "推荐景点" / "美食推荐" / "拍照打卡点"
- "X天X城市旅游" / "自由行" / "穷游"
- "旅游路线" / "行程安排" / "旅行预算"

## 三、技术架构

### 3.1 工具链集成

| 能力 | 工具/平台 | 集成方式 | 状态 |
|------|----------|---------|------|
| 火车票查询 | 12306 MCP Server | `npx -y 12306-mcp` | ✅ 已验证 |
| 地图+路径 | 高德地图 Web服务API | REST API | ✅ 可用 |
| 地图MCP | 高德地图 MCP Server | SSE直连 | ✅ 可用 |
| POI/天气 | 高德地图 API | REST API | ✅ 可用 |
| 旅游攻略 | 小红书MCP | `npx -y xhs-mcp` | 🔶 推荐启用 |
| 机票查询 | Web搜索 | 携程/飞猪/去哪儿 | ✅ 可用 |
| 酒店/民宿 | Web搜索 | 全平台比价 | ✅ 可用 |
| 报告生成 | 内联HTML模板 | 自研 | ✅ 内置 |

### 3.2 数据流

```
用户输入（城市/天数/兴趣/预算/同伴）
    │
    ├─→ 1. 景点推荐（核心）
    │      ├─→ 高德POI搜索（类型：风景名胜/公园/博物馆）
    │      ├─→ 小红书MCP → 热门攻略/打卡点
    │      └─→ Web搜索 → 评分/门票/开放时间
    │
    ├─→ 2. 打卡路线优化
    │      ├─→ 获取景点坐标
    │      ├─→ 高德路径规划 → 计算景点间距离/时间
    │      └─→ AI排序 → 最优游览顺序（地理位置就近原则）
    │
    ├─→ 3. 美食推荐
    │      ├─→ 高德POI → 当地特色餐厅
    │      ├─→ 小红书 → 网红美食/必吃榜
    │      └─→ 输出：店名+招牌菜+人均+位置
    │
    ├─→ 4. 拍照点推荐
    │      ├─→ 小红书 → 网红拍照地
    │      ├─→ 高德 → 最佳观景点
    │      └─→ 输出：位置+最佳时间+拍摄建议
    │
    ├─→ 5. 住宿推荐（全平台比价）
    │      ├─→ 高德POI → 民宿/酒店
    │      ├─→ Web搜索 → 各平台价格
    │      └─→ 民宿优先（爱彼迎/途家）+ 酒店备选
    │
    ├─→ 6. 交通方案
    │      ├─→ 12306 MCP → 火车/高铁
    │      ├─→ 高德路径 → 自驾/公交
    │      └─→ Web搜索 → 航班
    │
    ├─→ 7. 预算计算
    │      └─→ 交通+住宿+餐饮+门票+购物 → 总预算
    │
    └─→ 8. HTML报告生成
           └─→ 精美HTML（打卡地图+美食清单+预算+预订链接）
```

## 四、执行流程 (SOP)

### Phase 1: 需求理解

| 参数 | 说明 | 示例 |
|------|------|------|
| 出发城市 | 出发地 | 北京 |
| 目的城市 | 旅游目的地 | 成都 |
| 旅行天数 | 行程天数 | 4天3晚 |
| 出行日期 | 预计出行时间 | 2026-05-01 |
| 同伴 | 独自/情侣/家庭/朋友 | 情侣出游 |
| 兴趣偏好 | 景点类型偏好 | 美食/文化/自然/拍照/夜生活 |
| 预算范围 | 旅行预算 | 穷游/舒适/奢华 |
| 住宿偏好 | 酒店/民宿/青旅 | 民宿优先 |
| 交通偏好 | 飞机/高铁/自驾 | 不限 |

### Phase 2: 数据采集

**⚠️ 数据真实性规则（核心红线）**：
1. **景点门票/开放时间 → 通过Web搜索实时获取**，严禁编造
2. **餐厅信息/人均消费 → 通过高德POI或小红书实时查询**
3. **酒店价格 → 通过携程/飞猪Web搜索实时获取**
4. **交通班次 → 通过12306 MCP实时查询**
5. **天气 → 通过高德天气API实时获取**
6. **如实时数据获取失败 → 明确告知用户"请点击链接查看"，绝不编造**
7. **预算计算 → 基于实时查询结果**，参考档位仅用于框架，不得当作实际价格

**2.1 景点推荐**
```
工具优先级:
  1. 高德POI搜索（类型：风景名胜/公园/博物馆/游乐场）
  2. 小红书MCP → 热门攻略/打卡点/评分
  3. Web搜索 → 门票价格/开放时间/游玩时长

对每个景点输出:
  - 名称 + 类型 + 评分
  - 门票价格 + 开放时间 + 建议游玩时长
  - 坐标（用于路线优化）
  - 推荐指数 + 亮点标签
```

**2.2 打卡路线优化**
```
算法:
  1. 获取所有景点坐标
  2. 高德路径规划 → 计算景点间距离/时间
  3. 就近原则排序 → 减少路程时间
  4. 考虑开放时间 → 合理安排先后
  5. 每天行程控制在8-10小时

输出: 每日打卡路线时间线
```

**2.3 美食推荐**
```
策略:
  1. 高德POI → 目的地特色美食（类型：餐饮服务/中餐厅/火锅等）
  2. 小红书 → 当地必吃榜/网红餐厅
  3. 按区域分组 → 对应每日行程

输出: [{name, cuisine, signature_dish, avg_price, rating, location}]
```

**2.4 预算计算**
```
三档标准:

| 项目 | 穷游 💚 | 舒适 💛 | 奢华 ❤️ |
|------|--------|--------|--------|
| 住宿（/晚） | ¥100-200 | ¥300-600 | ¥800-2000 |
| 餐饮（/天） | ¥80-120 | ¥200-350 | ¥500+ |
| 门票（/天） | ¥0-100 | ¥100-300 | ¥300+ |
| 交通（/天） | ¥30-50 | ¥50-100 | ¥100-200 |
| 购物/娱乐 | ¥50 | ¥200 | ¥500+ |
```

### Phase 3: 行程编排

```
规则:
1. 每天安排3-5个景点（视距离而定）
2. 上午/下午各安排1-2个景点
3. 午餐安排在景点附近
4. 晚餐推荐当地特色餐厅
5. 拍照点穿插在景点之间
6. 留出弹性时间（购物/休息）
```

### Phase 4: 多格式报告生成

```
输出格式: HTML + PDF + 飞书云文档（按平台能力自动选择）
文件命名: {destination}_leisure_plan_{date}.html

格式选择逻辑:
  1. 有 feishu MCP → 额外生成飞书云文档
  2. 有 write_to_file → 生成 HTML + 预览
  3. 用户要求PDF → 生成 PDF
  4. 都没有 → 输出 Markdown 文本

报告结构:
  1. Hero区: 出行概述（目的地+日期+主题）
  2. 景点推荐打卡路线（含门票/开放时间）
  3. 美食深度推荐（含人均消费）
  4. 拍照打卡点（含最佳时段/角度）
  5. 住宿推荐（全平台比价）
  6. 交通方案
  7. 预算明细
  8. 天气穿搭
  9. 旅行清单

交付:
  - HTML报告（默认，preview_url 预览）
  - PDF（用户要求时）
  - 飞书云文档（有飞书MCP时）
  - deliver_attachments 回传
```

报告结构:
  1. Hero区: 旅行主题（城市+天数+风格标签）
  2. 每日打卡路线时间线（含地图标注）
  3. 景点推荐详情（评分/门票/时长/亮点）
  4. 美食推荐清单（店名+招牌菜+人均）
  5. 拍照打卡点（位置+最佳时间+建议）
  6. 住宿推荐（全平台比价+预订链接）
  7. 交通方案对比
  8. 预算明细表
  9. 天气+穿搭建议
  10. 旅行清单（行李/证件/药品/电子设备）
```
